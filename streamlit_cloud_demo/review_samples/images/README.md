# review_samples/images/ — 示例图片目录

将 NEU-DET 六类缺陷样本图片放入此目录，即可在检测页通过下拉菜单直接选择测试。

## 六类缺陷
- crazing (龟裂)
- inclusion (夹杂)
- patches (斑块)
- pitted_surface (麻点)
- rolled_in_scale (氧化皮)
- scratches (划痕)

## 图片来源
可从 NEU-DET 公开数据集 (https://www.kaggle.com/datasets/kaustubhdikshit/neu-surface-defect-database) 下载样本图片放入此目录。

## 支持格式
JPG / JPEG / PNG / BMP
